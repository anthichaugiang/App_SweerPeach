### Sweet Peach Project Setup Guide

1. **Set Up IP Address:**
   - Open the `application.yml` file in the backend and `/apis/api_const.dart` in the frontend.
   - Replace the IP Address with the IP address of the server you are using.

2. **Import SQL into phpMyAdmin:**
   - Open a web browser and navigate to phpMyAdmin.
   - Create a new database named `sweet_peach`.
   - Import the SQL file into the `sweet_peach` database.

3. **Run Backend:**
   - Open IntelliJ IDEA and navigate to the `sweet_peach_be` directory.
   - Configure and run the backend on IntelliJ IDEA.

4. **Run Frontend:**
   - Open Android Studio and navigate to the `sweet_peach_fe` directory.
   - Configure and run the frontend on Android Studio.

5. **Connect Android Device via USB:**
   - Connect your Android device to your computer using a USB cable.
   - Ensure that "USB Debugging" feature is enabled on your Android device.

6. **Run the App on Android Device:**
   - Run the app on your Android device from Android Studio.

Now your project is deployed and running on your development environment. Good luck!
